# Copy Weight Same Name Object

1. ウェイトバインディングされた子を持つオブジェクト 1 を選択
1. ウェイトバインディングされた子を持つオブジェクト 2 を選択
1. スクリプトを実行
1. オブジェクト 1 の子オブジェクトと同じ名前を持つオブジェクト 2 の子オブジェクトへウェイトコピーされる

※ 階層が違っても同じ名前の一番上にあるオブジェクトへウェイトコピーされるので注意
